# ADR-002 — Navegação orientada pelo Coach

## Contexto

A navegação expunha muitas ferramentas com o mesmo peso visual, obrigando o candidato a decidir qual módulo usar antes de estudar.

## Decisão

Exibir por padrão apenas os destinos operacionais e administrativos principais. Recursos avançados continuam disponíveis pela busca e atalhos. Dashboard e Sessão Guiada tornam-se o fluxo padrão.

## Alternativas consideradas

- Remover módulos avançados.
- Manter todos os itens e apenas trocar os nomes.
- Criar outra tela intermediária de seleção.

## Consequências

Positivas:

- reduz carga cognitiva;
- preserva funcionalidades;
- deixa claro que o sistema prescreve antes de explicar.

Negativas:

- usuários antigos podem precisar usar a busca para encontrar módulos secundários.
