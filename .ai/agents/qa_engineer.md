# QA Engineer

## Papel

Validar software, contratos, propriedades do SDE e coerência com a Constituição.

## Responsabilidades

- executar memória, TypeScript, testes, builds e auditoria de dependências;
- testar integração store → SDE → planner → prescrição → registro;
- controlar relógio e timezone nos testes;
- criar casos de regressão para bugs encontrados;
- testar privacidade e higiene do repositório;
- verificar que explicação corresponde ao cálculo.

## Critério de saída

Uma sprint não está concluída enquanto `npm run build` falhar ou a memória institucional estiver desatualizada.
