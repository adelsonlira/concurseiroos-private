# Curadoria das 37 provas FGV

Arquivos principais:

- `fgv37-curation-status.json`: estado consolidado das três ondas.
- `fgv37-review-required-priority.csv`: fila operacional das 383 questões únicas ambíguas.
- `FGV37_REVIEW_REQUIRED_PLAN.md`: regras da fila de revisão.
- `fgv37-gabarito-request-manifest.csv`: ordem recomendada para localizar gabaritos.
- `FGV37_GABARITO_REQUEST_PLAN.md`: justificativa do pedido gradual de gabaritos.
- `wave1/`, `wave2/`, `wave3/`: corpus, resumo, fila e auditorias de cada onda.

Nenhum arquivo deste diretório está autorizado a alterar o SDE enquanto a fonte estiver `RAW_UNCURATED`.
