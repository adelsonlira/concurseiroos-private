# Release notes — ConcurseiroOS 3.21.0

## Principal mudança

O sistema passa de prescrição executável para ciclo pedagógico fechado: o Coach orienta a sessão, coleta recuperação antes/depois e decide automaticamente a próxima ação.

## Novas capacidades

- ledger auditável de curadoria;
- taxonomia oficial e cobertura do edital;
- propostas conservadoras de classificação;
- matriz histórica estritamente shadow;
- fechamento guiado de aprendizagem;
- comando operacional único do Coach;
- estabilidade de ranking;
- prontidão pública e sem segredos;
- onboarding com padrões seguros.

## Compatibilidade

Snapshots passam a incluir `evidenciasAprendizagemGuiada`. Campos anteriores são preservados e a hidratação aceita backups sem esse campo.

## Segurança estratégica

Nenhuma classificação automática ou estatística histórica foi ativada no SDE. O produto não declara aprovação inevitável.
