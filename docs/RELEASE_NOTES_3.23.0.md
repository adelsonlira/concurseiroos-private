# Notas da versão 3.23.0

## Para o estudante

- `Recalcular` agora se chama `Atualizar recomendação` e explica que não altera histórico.
- `Plano e Progresso` mostra `Hoje` no comando que retorna a data de referência.
- A sincronização normal é automática; `Verificar agora` é apenas uma ação manual de diagnóstico.
- Conflitos aparecem somente quando dispositivo e nuvem receberam alterações concorrentes.
- O cofre privado é agrupado por disciplina e assunto.
- Perguntas-guia são respondidas antes e depois da sessão, no próprio fluxo da Sessão Guiada.
- O reset baixa backup, preserva a nuvem e restaura somente o dispositivo atual.
- Avisos metodológicos foram separados de falhas bloqueantes.

## Para manutenção

- sincronização baseada em revisão remota, fingerprint local e última base comum;
- reset de metadados cloud e vínculos IndexedDB;
- novo contrato de rascunho e resposta guiada;
- ADR-021 e ADR-022 registram as decisões.
